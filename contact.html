Frequently Asked Questions

What is ContactHub?

ContactHub is a cloud-based Contact Management System that allows users to securely store, manage and organize their contacts using Firebase.

Is my information secure?

Yes. User authentication is handled using Firebase Authentication, while contact information is securely stored using Firebase Firestore.

Can I edit and delete contacts?

Absolutely. Every registered user can create, read, update and delete their own contacts at any time.

Is ContactHub free?

Yes. ContactHub is developed as an educational and portfolio project to demonstrate modern web development concepts.
